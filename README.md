# Jenkins + Docker CI/CD Project 🚀

## 📌 Overview
This project demonstrates a simple CI/CD pipeline using Jenkins and Docker.

## 🔧 Tools Used
- Jenkins
- Docker
- GitHub
- Nginx

## ⚙️ Workflow
1. Pull code from GitHub
2. Build Docker image
3. Login to Docker Hub
4. Push image to Docker Hub
5. Run container

## 🌐 Output
Application runs on:
http://<server-ip>:and the port your host exposed in my case its 56565

## 🧠 Key Learnings
- Jenkins Declarative Pipeline
- Docker build & push
- Credential management in Jenkins
- Handling real-world errors (permissions, ports)
